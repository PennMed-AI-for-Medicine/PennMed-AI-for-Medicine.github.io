# PennMed-AI-for-Medicine.github.io

Course webpage for MPHY 6120: Artificial Intelligence for Medicine (Spring 2025)

## Course Modules

| Module | Title | Topics |
|--------|-------|--------|
| 0 | Course Introduction & Python Foundations | Environment setup (uv), NumPy, Pandas, PyTorch basics |
| 1 | Medical Imaging Data & DICOM | DICOM standard, pydicom, medical image visualization |
| 2 | Statistics, EDA & Linear Algebra | Exploratory data analysis, statistical foundations |
| 3 | Traditional Machine Learning | Regression, classification, model evaluation, clustering |
| 4 | PyTorch & Introduction to Deep Learning | Neural networks, autograd, training loops |
| 5 | Computer Vision for Medical Imaging | CNNs, U-Net segmentation, transfer learning |
| 6 | NLP & Language Models for Clinical Text | Tokenization, transformers, LLMs in medicine |
| 7 | Advanced Topics & Course Wrap-up | RL, ethics, FDA regulations, deployment |

## Homeworks

- **HW1**: Python Environment & Basics (Modules 0-1)
